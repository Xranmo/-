# 项目描述
### 新一轮电力市场自2015年启动以来迄今已有5年时间。本轮电力市场改革旨在开放配售段自由交易，改革的力度空前，而改革的措施也需要深入研究，改革的市场也必将持续十年甚至数十年。以四川电力市场为例，基本保持着市场规则一年一大变，几月一小变的改革步伐，因此对于参与电力市场交易的企业来说，常常因为规则的频繁变动而摸不着头脑。本项目旨在对四川电力市场上半年的交易情况进行梳理，采用可视化的方式为企业提供更直观的交易情况展示，为企业更好地做出交易决策提供支撑。
